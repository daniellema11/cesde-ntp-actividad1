# Actividad Práctica: Fundamentos de Python

Bienvenido a la actividad de ejercicios de Python. En la carpeta `ejercicios` encontrarás una serie de archivos `.py` secuenciales diseñados para poner a prueba y reforzar tus conocimientos básicos de programación.

## Instrucciones

Para completar esta actividad correctamente, debes seguir estos pasos:

1.  **Haz un Fork del repositorio:** En la esquina superior derecha de esta página, haz clic en el botón "Fork" para crear una copia de este repositorio en tu propia cuenta de GitHub.
2.  **Clona tu repositorio:** Clona tu nuevo repositorio (el fork) en tu máquina local.
3.  **Abre tu entorno de desarrollo:** Inicia tu editor de código o IDE preferido (como VS Code) y abre la carpeta de este proyecto.
4.  **Sigue el orden numérico:** Abre los archivos dentro de la carpeta `ejercicios` comenzando desde el `ejercicio_01.py` hasta el `ejercicio_10.py`.
5.  **Lee los enunciados:** Dentro de cada archivo encontrarás comentarios explicativos que detallan el tema y los retos a solucionar.
6.  **Escribe tu código:** Implementa tus soluciones al final de cada archivo, estrictamente debajo de la línea que dice `# Escribe tu código debajo de esta línea:`.
7.  **Prueba tu solución:** Ejecuta cada archivo desde la terminal (por ejemplo, escribiendo `python ejercicios/ejercicio_01.py`) para verificar que el programa funciona y muestra los resultados correctos.

**⚠️ Importante: Evaluación**
Ten en cuenta que **se evaluarán los ejercicios comentados dentro de cada archivo**. Asegúrate de probar todas tus soluciones en cada `.py` antes de enviar o confirmar tu entrega final.

## Lo que hay que hacer (Objetivos por Archivo)

A continuación se describe el trabajo que debes realizar en cada uno de los archivos:

*   **`ejercicio_01.py` (Python Básico):** Tu tarea es utilizar la función `print()` de diferentes maneras para mostrar mensajes y palabras en la consola.
*   **`ejercicio_02.py` (Variables):** Debes declarar variables (nombres, precios, edades), asignarles los valores indicados e imprimir los resultados en pantalla.
*   **`ejercicio_03.py` (Operadores):** Tienes que resolver operaciones aritméticas básicas utilizando código (sumas, restas, multiplicaciones, divisiones y cálculo de residuos).
*   **`ejercicio_04.py` (Condicionales):** Deberás crear estructuras lógicas (`if`, `elif`, `else`) para comprobar si los números son positivos, mayores de edad o pares, imprimiendo el mensaje correspondiente.
*   **`ejercicio_05.py` (Ciclos):** Tu trabajo es configurar bucles `for` y `while` que automaticen el conteo de secuencias de números y realicen una suma acumulativa.
*   **`ejercicio_06.py` (Colecciones):** Debes construir y mostrar las 4 estructuras de datos fundamentales de Python: una lista (colores), una tupla (coordenadas), un conjunto (frutas) y un diccionario (datos de un libro).
*   **`ejercicio_07.py` (Input y Casting):** Deberás crear programas interactivos que pregunten datos al usuario, los conviertan a números (enteros o decimales) y apliquen cálculos con ellos.
*   **`ejercicio_08.py` (Iteración de Colecciones):** Tu objetivo es combinar ciclos con listas y diccionarios para recorrer sus elementos individualmente, aplicarles condiciones y generar nuevas listas.
*   **`ejercicio_09.py` (Input Avanzado y Formateo):** Tienes que solicitar múltiples datos al usuario, combinarlos en frases interactivas usando formateo de cadenas y practicar la multiplicación de textos.
*   **`ejercicio_10.py` (Funciones):** Debes definir bloques de código reutilizables (`def`) que reciban parámetros, procesen información (sumas o validaciones de números pares) y retornen el resultado al programa principal.
